# VikingExpress Logistics Manager

VikingExpress is a Java-based logistics management system designed to streamline shipment tracking and fleet coordination. This project focuses on implementing robust business logic and efficient data structures to handle logistics workflows without the overhead of a persistent database.

## Key Features
* **Shipment Coordination:** Logic for managing and routing packages through various logistical nodes.
* **In-Memory Data Management:** Efficient handling of shipment and fleet data using Java Collections Framework.
* **Terminal-Based Interface:** A streamlined Command Line Interface (CLI) for real-time interaction and system management.
* **Logistics Validation:** Built-in rules for weight limits, delivery scheduling, and status updates.

## Technical Stack
* **Language:** Java 8+
* **Build Tool:** Maven
* **Architecture:** Object-Oriented Design focusing on separation of concerns (Models, Services, and Controllers).
